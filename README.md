# 📊 Excel Practice & Projects Repository

This repository contains my complete learning journey in Microsoft Excel for Data Analysis.

It includes practice datasets, formula exercises, mini projects, full dashboard projects, and structured notes covering important Excel concepts required for a Data Analyst role.

## 📁 Repository Contents

### ✅ Practice Files

* Basic Excel formula practice (SUM, AVERAGE, COUNT, IF)
* Lookup functions practice (VLOOKUP, XLOOKUP, INDEX + MATCH)
* Conditional functions (SUMIF, COUNTIF, nested IF)
* Data cleaning exercises (TRIM, TEXT functions, date formatting)
* Pivot table practice datasets

### 📊 Mini Projects

* Sales data analysis
* Region-wise revenue calculation
* Product performance comparison
* Monthly trend analysis using Pivot Tables
* Data cleaning and transformation tasks

### 📈 Dashboard Projects

* Interactive Sales Dashboard using Pivot Tables and Charts
* KPI cards (Total Revenue, Orders, Average Sales)
* Slicer-based filtering for dynamic analysis
* Trend visualization using column and line charts

### 📝 Notes

* Excel formulas explanation with examples
* Pivot Table concepts and use cases
* Data cleaning best practices
* Dashboard design principles
* Interview preparation notes for Excel

## 🎯 Objective

The goal of this repository is to build strong Excel skills required for Data Analyst roles by practicing real-world datasets and creating meaningful dashboards.

This repository reflects my hands-on learning and continuous improvement in data analysis.
